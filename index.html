Personal Home/Office Automation Monkey

I'm slowly teaching my monkey to do all the chores that I don't like. Every now and then I teach it a new trick, but for now, here's what it can do...

Accounting

  • reads and manipulates human-readable ledger files
  • imports bank statements in CSV format into my accounting ledger

Banking

  • does online-banking via HBCI (using aqbanking-cli)
    • checks account balances
    • issues transfers

Small-business management

  • keeps track of my customers and suppliers
  • generates time-sheets for contract work
  • generates invoices ready to send

E-mail processing

  • received invoices to account for business expenses
  • time-recording reports for invoice generation

Note: Not everything is already published, some things are still implemented as crude Rake scripts, using "monkey" only as a Ruby library.

Usage

My monkey's primary interface is a self-documenting command-line application, but it can also be used directly from Ruby as library. I personally prefer to use pry during development and when I need something from my monkey that can't be requested yet via the command-line interface.

Here's how to get started:

bundle install
bundle exec bin/monkey
